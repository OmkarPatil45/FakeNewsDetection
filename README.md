# Fake News Detection (or relevant title)

This project uses Machine Learning to classify data based on a Linear Regression model. It includes a trained model and a vectorizer for processing input text.

## 📂 Folder Structure
* `app.py`: The main Python application script.
* `app.ipynb`: Jupyter Notebook for experimentation.
* `lr_model.jb`: The saved Linear Regression model.
* `vectorizer.jb`: The saved text vectorizer.
* `Fake.csv` / `True.csv`: Datasets used for training/testing.
